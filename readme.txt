== Description ==

This plugin gives you making QR Code for your article permalink with Google Chart request in EASY. You are able to write shortcode like [qrcode-gc alr:"altname" class:"classname" id:"idname"] . If you require to apply new size QRCode, you might remove wp-qrcode-gc directory.

== Installation ==

1. Upload `wp-qrcode-by-google-chart` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'Settings' link in the plugins row to configure the plugin
4. You put short code ‘[qrcode-gc]’ option: id, class, alt
